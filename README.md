# Unify
Asp.Net Core and AngularJS RSS Reader 

#Features
- Rss channel as article source support
- Dynamic article sources navigation and CRUD (SPA)
- Source tags support
- Simple filters (all, unread, saved)
- Marking articles as read
- Simple article search
- Saving articles
